<b>ACTIE </b>- Deze functie moet een heldere definitie hebben, met bijvoorkeur een bronvermelding. 1.Wettelijke definities 2. Defnities uit normen 3. NVS definites 4. Begrippenlijst Nictiz.<br/><br/><font color="#dc143c"><b>RISICO </b></font>– Zonder een heldere en eenduidige definitie van deze functie ontbreekt een gezamenlijk begrip, waardoor betrokkenen niet over hetzelfde spreken.<br/><br/>Opmerking - voor de definitie zal ik in eerste instantie gaan zoeken in NEN7517 (Lokalisatie)<br/><br/><font color="#ff6347"><b>POGING </b></font>- De Adressering functie voegt een digitaal uniek adres van een zorgaanbieder toe aan een bericht, zodat deze afgeleverd kan worden op de juiste plek.
